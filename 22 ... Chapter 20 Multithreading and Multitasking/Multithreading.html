Global Variables All threads see and can modify global data. Requires synchronization (Mutex/CriticalSection).
Heap Memory Pointers from malloc or new are accessible by everyone if the address is shared.
System Resources Open files, kernel handles, and Window objects (HWND) belong to the process.
The Stack Each thread has its own function call history, return addresses, and local auto variables.
CPU Registers Context includes the Instruction Pointer (EIP/RIP) and Stack Pointer (ESP/RSP).
Thread Local Storage (TLS) Special __declspec(thread) variables that are unique instances for every thread.
