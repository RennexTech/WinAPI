Common Virtual Keys (VK_ Codes)

The VK_ codes are special numerical identifiers for physical keys on your keyboard. When a user presses a key, the system sends a message to the program indicating which key was pressed. This message includes a wParam value that is usually the VK_ code for the physical key. These codes are used in programming to detect key presses and perform corresponding actions like opening menus, submitting forms, or controlling games. The key difference between WM_KEYDOWN (which detects the physical key) and WM_CHAR (which detects the actual character typed) is important when you're working with events in a program.

Here’s a cheat sheet for some of the most common VK_ codes that you might encounter:

Virtual Key Code What It Means (The Key) Use Case Vibe
VK_ESCAPE The 'Esc' key "Nah, cancel that." - Often used to exit or cancel operations like closing windows or quitting games. It's the ultimate "nope" key, especially useful for canceling or backing out of something.
VK_RETURN The 'Enter' key "Let's go!" - Used for submitting forms, confirming choices, or creating a new line in a text input. It’s a key that signals "go ahead" or "confirm."
VK_SPACE The 'Spacebar' "Jump!" or "Pause." - The spacebar is vital in games for actions like jumping or pausing. It's also the key to typing spaces between words. It toggles a lot of actions beyond just typing!
VK_BACK The 'Backspace' key "Oops, erase that." - Used for deleting characters or items in text input fields, this key is your "undo" in typing.
VK_TAB The 'Tab' key "Next field, please!" - It helps you navigate through different fields in forms or text boxes. It's often used for moving from one area to another without using the mouse.
VK_DELETE The 'Delete' key "Poof, it's gone!" - Used for removing selected items or characters after you’ve highlighted them. It's another way to erase content.
VK_LEFT Left Arrow key "Go left!" - Used for moving left, navigating menus, or in games to control movement of a character or object.
VK_UP Up Arrow key "Go up!" - This key is used for navigating upwards through menus, scrolling up, or moving a cursor or character upwards.
VK_RIGHT Right Arrow key "Go right!" - Similar to the left arrow, but this key moves your selection or character to the right.
VK_DOWN Down Arrow key "Go down!" - Used for navigating down through menus, scrolling down, or moving characters downward in games.
VK_SHIFT Either Shift key "Hold on to this." - Often paired with other keys to perform special actions, such as creating capital letters or activating shortcuts like Shift + Click for extra functionality.
VK_CONTROL Either Ctrl key "Power move!" - A key that gets paired with others for common actions like Ctrl + C to copy or Ctrl + V to paste. It’s the foundation of many keyboard shortcuts!
VK_MENU Either Alt key "Menu time!" - Often used to open menus or perform actions like Alt + F to open the File menu. It’s another key that powers lots of shortcuts.
VK_F1 to VK_F24 Function keys (F1, F2, etc.) "Shortcut central!" - These function keys are often linked to specific actions, like F1 for Help, F5 for Refresh, or F12 for Developer tools in browsers.
VK_0 to VK_9 Number keys (0-9) "Counting on it." - Used to type numbers directly. These keys are essential for data entry, calculations, and more.
VK_A to VK_Z Letter keys (A-Z) "Typing it out." - These are the basic alphabet keys that you use to type out words. Each key represents a letter from A to Z.
VK_ADD Numpad '+' key "Add it up!" - For performing addition tasks in calculators or other mathematical programs.
VK_SUBTRACT Numpad '-' key "Take it away." - For performing subtraction in math-related tasks.
VK_MULTIPLY Numpad '*' key "Times that!" - Used for multiplication in calculators or other mathematical inputs.
VK_DIVIDE Numpad '/' key "Split it evenly." - Used for division in mathematical tasks.
VK_OEM_1 Semicolon (';') key "Punctuation check!" - This key varies depending on the keyboard layout, but it’s often associated with typing punctuation marks.
VK_OEM_2 Question mark ('?') key "What's up?" - A special key that’s used for typing the question mark, depending on your layout.
VK_OEM_3 Tilde ('~') key "Wavy, dude." - Often used for typing the tilde (~) symbol or opening developer consoles in games or browsers.
